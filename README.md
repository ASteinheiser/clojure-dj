# clojure-dj

A Clojure app, made with Overtone, designed to make (good?) music...

## Install dependencies
```
lein install
```

## Compile files
```
lein compile
```

## Run locally
```
lein run
```

## Build app
```
lein uberjar
```

## Run compiled app
```
java -jar target/clojure-dj-0.1.0-SNAPSHOT-standalone.jar
```
